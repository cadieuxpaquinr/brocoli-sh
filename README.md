# brocoli-sh

The **brocoli shell** stands for Brocoli Rocoli Ocoli Coli Oli Li I SHell.

It is a shell meant to be better for your health than other shells.

# Build & run
```
cmake -S . -B build
cmake --build build
build/bin/bro
```
